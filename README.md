Quick Start
-----------

1) Requirement

	You will need :
	- Scala
	- Apache Maven

2) Start

	Run command mvn compile to compile the project. If you are using Eclipse, run mvn eclipse:eclipse to make eclipse maven-friendly, then run mvn compile
	Run XXX.scala as a scala application
